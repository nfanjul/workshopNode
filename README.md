# workshopNode
Workshop nodejs

1. Installation
Install all the dependencies for the project.
npm install

2. Develop
Runs a development server at https://localhost:1337.
npm start

Important: COnfigure the connection for your bd.:
var config = {
    user: 'xxxxx',
    password: 'xxxxx',
    server: 'xxxxx',
    database: 'xxxxx',
    options:
    {
        encrypt: true,
    }
}
